各种效果的轮播图

三种常见的轮播图特效

  1. 滑动的轮播图
  2. 透明度变化的轮播图
  3. 层叠样式的轮播图 ###　滑动的轮播图
    特效展示： 滑动效果　　

主要思路：　　

  • 还是利用一个小的可视区域包住一个大的列表。　　
  • 然后让那个大的列表的间距为负值，慢慢划出可视区域。
  • 最后将列表第一个加入到列表尾部，并将间距设为０

  • 优点：不用像以前的利用绝对定位改变ｔｏｐ值或者ｌｅｆｔ值来实现滑动特效，也不用判断高度或者宽度。

    setInterval(function(){
    var aLi=$('#pic_list li');
    $('#pic_list').animate({ marginLeft: -202},800,function(){
        aLi.eq(0).appendTo('#pic_list');
        $('#pic_list').css('marginLeft','0');
    });
},1800);

###　透明度改变的轮播图
效果展示：渐入渐出效果　　

主要思路：　　

  • 将需要展示特效的图片利用定位层叠在一起
  • 然后将当前的图片的定位层级设为最高，透明度慢慢变为０，这样，下层图片也会慢慢显现出来
  • 让透明度变为０的图像的透明度变为１，同时将定位层级设为最低。
  • 重复操作　

  • 优点：　直接用ｆａｄｅＩｎ图片的特效是先慢慢变黑然后再慢慢显示另一张

    setInterval(function(){
    fade(aImg.eq(i));

    i--;
    aImg.eq(i).css('opacity','1');
    aImg.eq(i).css('z-index',i);
    if(i<0){
        i=2;
    }

},2000);

function fade(obj){
    obj.css('z-index','100');
    obj.animate({ opacity: 0 },2000);
}

###　层叠样式的轮播图
特效展示：　层叠轮播样式

主要思路：

  • 首先将图片按照静态样式定位好
  • 每一个图片都有自己的宽高和定位的ｔｏｐ，ｌｅｆｔ，透明度，定位层级等等
  • 当动画进行时，将每一个图片的样式设置保存为一个数组，然后切换数组
  • 再将切换完成的数组内的样式重新设置到图片上
  • 切换数组也就是将第一个放到最后一个，或者最后一个挪到第一个的位置

  • 创新思路：　将样式设置保存到数组中，通过切换数组来切换图片。

    for(var i=0;i<aLi.length;i++){
    arr.push([ parseInt(getStyle(aLi[i],'left')), parseInt(getStyle(aLi[i],'top')), getStyle(aLi[i],'z-index'), getStyle(aLi[i],'opacity')*100, aImg[i].width, aImg[i].height]);
}

prev.onclick=function(){
    arr.unshift(arr[arr.length-1]);

        aLi[i].style.zIndex=arr[i][2];

next.onclick=function(){

    }
}
