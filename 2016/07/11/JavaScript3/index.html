DOM事件的相关方法及属性

DOM定义

DOM : Document Object Model 文档对象模型
DOM会把文档看作是一棵树，同时定义了很多方法来操作这棵数中的每一个元素（节点）

DOM相关方法及属性

(1).firstChild:

标准下：firstChild会包含文本类型的节点  
非标准下：firstChild 只包含元素节点  
firstElementChild: 标准IE下的第一个元素节点，非标准下没有这个属性

(2) 子节点和兄弟节点

元素.firstElementChild || 元素.firstChild 第一个子节点  
元素.lastChild || 元素.lastElementChild 最后一个子节点  
元素.nextSibling || 元素.nextElementSibling 下一个兄弟节点  
元素.previousSibling || 元素.previousElementSibling 上一个兄弟节点  
都有一个问题：如果为空，则会是firstChild是一个文本元素，并不是我们想要的属性节点。  
解决方法1：第一个子节点可以用   元素.children[0]

(3)父节点

元素.parentNode : 只读 属性 当前节点的父级节点  
元素.offsetParent: 只读 属性 离当前元素最近的一个有定位属性的父节点  
如果没有定位父级，默认的是body  
ie7以下，如果当前元素没有定位，则默认是body，如果当前元素有定位，则默认是html  
ie7以下，如果当前元素的某个父级触发了layout，那么offsetParent就会指向触发了
        layout特性的父节点上。

(4)offsetLeft[top]

offsetLeft[top]: 只读 属性当前元素到定位父级的距离（偏移值），
也可以说：到当前元素的offsetParent的距离.

如果没有定位父级 

offsetParent -> body  
offsetLeft -> html

如果有定位父级

ie7以下：如果自己没有定位，那么offsetLeft[Top]是到body的距离  
    如果自己有定位，那么就是到定位父级的距离  
其他：到定位父级的距离

(5).width height

style.width : 样式宽      //100   
clientWidth : 可视区宽    //样式宽 + padding
offsetWidth : 占位宽      //样式宽 + padding + border

(6)获取元素的绝对位置：

function getPos(obj) {
    var pos = {left:0, top:0};  
    while (obj) {
        pos.left += obj.offsetLeft;
        pos.top += obj.offsetTop;
        obj = obj.offsetParent;
    }  
    return pos;   
}

(7)操作元素的属性：

元素.getAttribute(属性名称); 方法 获取指定元素的指定属性的值
元素.setAttribute(属性名称，属性值); 方法 给指定元素指定的属性设置值
元素.removeAttribute(属性名称); 方法 移除指定的元素的指定的属性

(8)元素的创建:

document.createElement(标签名称); 创建元素
父级.appendChild(要添加的元素) 方法 追加子元素
父级.removeChild(要删除的元素); 删除元素
父级.insertBefore(新的元素，被插入的元素) 方法 在指定元素前面插入一个新元素
    在ie下如果第二个参数的节点不存在，会报错
    在其他标准浏览器下如果第二个参数的节点不存在，则会以appendChild的形式进行添加
    (解决方法：用一个if语句判断，如果第一个为空，则用appendChild()添加元素)
父级.replaceChild(新节点，被替换节点) 替换子节点

appendChild insertBeforere placeChild 都可以操作动态创建出来的节点,也可以操作节点

BOM事件

Browser Object Model 浏览器对象模型
(1) open(地址默认是空白页面，打开方式默认新窗口) 打开一个新窗口 

opener = window.open('http://www.baidu.com', '_self');

(2) window.close(); ff : 无法关闭 chrome : 直接关闭 ie : 询问用户
(3) opener.close(); //可以关闭在本窗口中通过js打开的新窗口
(4) window.navigator.userAgent : 浏览器信息
(5) window.location : 地址 

window.location.href = window.location内容
window.location.search = url?后面的内容
window.location.hash = url#后面的内容
